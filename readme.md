# Task 2: Data Visualization & Storytelling

## Project Title
Product Sales Insights Dashboard

## Overview
This project presents a sales dashboard built using an Excel dataset. The dashboard helps analyze product sales, profit, region performance, monthly trends, and sub-category contribution through interactive visualizations.

## Tools Used
- Microsoft Excel
- Power BI / Tableau
- GitHub

## Dataset Used
The dataset contains the following fields:
- Product Name
- Order ID
- Quantity
- Price
- Category
- Date
- State
- Region
- Sub-Category
- Sales
- Profit

## Objective
The objective of this task is to clean the data, create meaningful charts, and use storytelling to explain business insights clearly.

## Dashboard Components
The dashboard includes:
- KPI Cards for total Sales and Profit
- Donut Chart for Profit by State
- Pie Chart for Sales by State
- Bar Chart for Profit by Product Name
- Line Chart for Sales by Month
- Map Chart for Sales by Country and Region
- Bar Chart for Sales by Sub-Category
- Region filter for interactivity

## Data Cleaning
Before creating the dashboard, the data was cleaned by:
- Removing duplicate records
- Handling missing values
- Formatting date columns properly
- Ensuring numeric columns were in the correct format
- Renaming fields where required

## Key Insights
- Texas has the highest sales among the states shown.
- Michigan contributes the highest profit in the profit chart.
- Sales vary significantly across months, with December showing the highest sales.
- Chairs, Binders, and Phones are the top-selling sub-categories.
- The region filter helps analyze performance by geography.

## Storytelling
The dashboard tells a clear business story by showing how sales and profit change across states, months, and sub-categories. It helps identify high-performing products and regions while also revealing seasonal sales trends.

## Deliverables
- Dashboard screenshot
- Cleaned dataset
- README.md file
- Optional report PDF

## Conclusion
This dashboard converts raw sales data into meaningful business insights. It demonstrates data cleaning, visualization, and storytelling in a clear and professional way.
